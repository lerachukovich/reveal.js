GitHub Features

Free and open source

- General Public License

- Absolutely free

- Ability to modify code as per your requirements

Small and fast

- Nearly all of operations performed locally

- Was built to work on the Linux kernel

- Written in C

Scalable

Git compresses all the huge data through a lossless compression technique.

Reliable

On the events of a system crash, the lost data can be recovered from any of the local repositories.

Secure

- SHA1 (Secure Hash Function)

- ID of a particular version depends upon the complete development history leading up to that commit.

- It is not possible to change the old versions without it being noticed.

Easy branching
